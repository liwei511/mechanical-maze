# Gobot 肉鸽游戏 - 简版像素肉鸽

基于 Go + goboy 引擎开发的简单肉鸽游戏。

## 核心概念

**"10层地牢，每一步都是新选择"**

极简像素风格，每次爬楼都是随机生成地图，捡装备打怪物，拿神器闯到顶层。

## 核心特色

- 🔀 **随机地图生成**：每次玩地图都不一样，真正肉鸽
- ⚔️ **简单战斗**：回合制，砍怪捡装备
- 🎒 **装备系统**：不同武器/护甲加成，随机掉落
- 🏆 **10层挑战**：从第一层打到第十层，打败最终Boss通关
- 🎮 **键盘控制**：方向键移动，空格攻击，简单操作
- **像素风格**：低分辨率像素，复古感觉

## 游戏流程

1. **出生在第一层楼梯口**
2. **探索地图**：扫开迷雾，找房间找楼梯
3. **打怪**：碰到怪物进入战斗，打死得金币和经验
4. **捡装备**：宝箱里开出更好的武器护甲
5. **找楼梯**：到下一层，难度递增
6. **商店**：偶尔遇到商店，用金币买血买装备
7. **第10层**：打最终Boss，通关

## 基础元素设计

### 地图
- 每层 20x20 网格
- 房间+走廊随机生成
- 开局全黑，玩家走到哪里亮哪里（迷雾探索）

### 玩家
- 生命值，攻击力，防御力
- 升级后三选一加成：+生命/+攻击/+防御

### 怪物
- 每层难度递增，怪物属性越来越高
- 不同外观：哥布林→兽人→骷髅→法师→恶魔
- 打死掉落金币和装备概率

### 物品
| 类型 | 作用 |
|------|------|
| 武器 | 加攻击 |
| 护甲 | 加防御 |
| 血瓶 | 恢复生命 |
| 金币 | 商店消费 |

### 特殊房间
- 普通房间：1-2只怪
- 宝箱房：无怪，直接开宝箱拿装备
- 商店：买血买装备
- 陷阱房：掉血或者出怪
- 楼梯：通往下一层

## 操作控制（键盘）
- `↑←↓→` 移动
- `空格` 攻击
- `i` 打开人物属性背包
- `z` 确认选择
- `q` 退出菜单
