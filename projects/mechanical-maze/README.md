# 机械迷城 (Mechanical Maze)

简版手绘风 2D 解谜微信小游戏，致敬经典《机械迷城》。

## 项目理念

保持机械迷城的核心蒸汽朋克机器人世界观，简化美术为手绘简版，保留经典解谜剧情节点，适合微信小游戏轻量化体验。

## 核心特色

- **蒸汽朋克机器人世界**：全机器角色，锈迹斑斑的机械城氛围
- **手绘简版美术**：简约线条，复古色调，保留手绘质感
- **经典解谜剧情**：保留机械迷城关键情节节点
- **触摸交互**：适配微信小游戏，点击/拖拽操作
- **无文字对话**：纯图标表情交流，保留经典特色

## 关键剧情节点（参考机械迷城）

1. **开头**：主角 Josef 被拆散扔到垃圾场，玩家帮助组装身体
2. **进城**：爬到机械城顶端，进入城市
3. **抓住三土匪**：三个坏机器人抢走了研究所的东西，需要设计抓住他们
4. **电梯故障**：修复电梯进入底层
5. **锅炉房**：点燃锅炉，打开通路
6. **时钟塔**：调整时钟打开大门
7. **研究所**：找到被囚禁的科学家，拿到枪
8. **最终Boss**：登上塔顶，打败大机器人，救出女朋友

## 特色系统

- **物品收集组合**：收集零件，组合道具解谜
- **迷你游戏**：在开锁等地方插入小解谜游戏
- **提示系统**：点击灯泡获取提示（经典苍蝇飞行提示）

## 技术栈

- 微信小游戏原生开发 / Cocos Creator
- 手绘美术资源
