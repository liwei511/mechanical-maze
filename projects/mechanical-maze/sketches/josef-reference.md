# 主角 Josef - 草图参考

## 比例造型

```
    O  ← 圆形大头 (直径约等于身体宽度)
   /|\ ← 细肩膀+短手臂
   ||| ← 胸腔 (圆柱形，比头窄一点)
   ||| ← 腹部 (比胸腔稍宽一点)
  /  \ ← 分叉细大腿
 /    \← 细小腿
```

- **头身比**：1:3，头大身体小，可爱比例
- 身体分三节：头 → 胸腔 → 下腹，关节处有圆形连接
- 四肢都是分节细棍，末端简化，没有明显手指脚趾

## 标志性特征（一定要保留）

1. 圆溜溜大脑袋
2. 细长分节身体
3. 没有嘴巴，只用眼睛眉毛表达表情
4. 头顶可以伸出一个小天线（原作的互动设计）

## 不同表情参考（线稿简化）

| 表情 | 画法 |
|------|------|
| 正常 | 两个小圆眼睛，平眉毛 |
| 开心 | 眉毛上扬，眼睛弯一点 |
| 疑惑 | 一个眉毛高一个低 |
| 惊讶 | 眼睛圆睁，眉毛抬起 |
| 难过 | 眉毛下撇 |

## 配色参考（上色时用）

- 脸部：浅灰 `#cccccc`
- 身体主体：锈铜 `#8b6e4b`
- 关节连接处：深棕 `#5a4530`
- 轮廓线：深褐 `#2d261e`
- 高光：浅铜 `#b89a6b`

## 草图绘制要点

1. 先画大圆定头部
2. 往下画细圆柱身体，分两节
3. 加四肢，都是细分节
4. 画表情，简单线条即可
5. 勾勒轮廓，加一点锈点斑点增加质感
