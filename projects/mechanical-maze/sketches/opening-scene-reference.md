# 开场场景 - 垃圾场 草图参考

## 场景概述

游戏开场，Josef 被拆散成零件扔在垃圾场，玩家需要把零件拼起来，这是第一个解谜。

## 整体布局

```
┌─────────────────────────┐
│  远处垃圾山  灰蒙蒙天空 │
│  · ˚ · 锈零件散落       │
│                         │
│  [头]  [身体]  [胳膊]   │
│  零件散落在地上         │
│                         │
│          地面           │
└─────────────────────────┘
```

- **视角**：横向2D场景，适合手机竖屏？不，横屏体验更好，微信小游戏支持横屏
- **层次**：远景垃圾山 → 中景散落零件 → 近景破碎地面

## 元素设计（简化版）

### 地面
- 破碎钢板拼接，有裂缝
- 几块大铁板错落地板，不平整
-  scattered 几个小螺丝、齿轮碎片点缀

### 散落的 Josef 零件
- 圆形大头 在左上方
- 身体两节 在中下方
- 两条胳膊 在右侧
- 两条腿 在左下方
- 零件都分开，玩家需要拖到正确位置拼接

### 背景垃圾山
- 堆积各种废弃机器零件
- 简化成不规则几何块：方形箱子、圆形齿轮、长条管道
- 不用每个都画清楚，概括体积感
- 整体比前景暗，营造深度

### 点缀
- 几根歪掉的生锈铁管立着
- 一两个破铜罐子斜倒着
- 远处飘着淡淡的灰烟（几个波浪线条）

## 光影氛围
- 整体偏暗，阴天光照
- 光源从上方来，每个零件底下加一块简单阴影色块
- 主角零件比背景亮一点点，突出可互动

## 配色参考
- 天空：淡灰蓝 `#7a8899`
- 远景垃圾山：深棕 `#4a3f33`
- 地面钢板：中棕 `#5a4a3a`
- 零件：和主角一致的锈铜色 `#8b6e4b`
- 阴影：深褐 `#2d261e` 半透明

## 草图要点
1. 先打透视框，确定地面倾斜角度（稍微一点仰视？不，平视就好）
2. 画远景垃圾山轮廓，概括形状
3. 安排主角零件位置，分散开但不要太乱
4. 加地面裂缝和小零件点缀
5. 勾轮廓，保留手绘感，不用太整齐
